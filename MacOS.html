Terminal in MacOS

Go to end of line: ctrl + l
Go to beginning of line : ctrl + a
Select and copy whole line : Two-fingered click
To close terminal on "Exit": Preferences → Shell → "When shell exits" → Close window

VIM

Go to end of line: $
Go to beginning of line : 0 (Zero)

matplotlib

By default, MacOS doesn't show the plots using plt.show().
Include following lines at the start:
include matplotlib matplotlib.use('TKAgg')
