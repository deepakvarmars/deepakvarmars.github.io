Using Spack

spack list : To list all available packages
spack find : To list all installed packages
spack install "package" : To install a package
spack spec "package" : To find the dependencies of a package with their version number
spack uninstall "package" : To uninstall a package
spack uninstall --all : To uninstall all installed packages

If there are any issues after installation, it is better to delete spack and restart as the cache contains all the downloaded scientific packages. So installation would give the exact same issues.

Mounting exFAT formatted HDDs on Linux machine

Since exFAT is the only unrestricted format that can be used on both macOS and Windows 10, it is good to change the format of an external HDD (using Disk Manager in Windows or Disk Utility in MacOS). However, this doesn't mount on Ubuntu Linux 17.10 in default settings. To access exFAT HDD in Linux, do the following:
sudo apt-get install exfat-fuse exfat-utils
sudo mkdir /media/exfat
sudo mount -t exfat /dev/sdxx /media/exfat

If you don't know if sdxx = sda1 or sdb1 etc, try:
sudo lsblk -o NAME,FSTYPE,SIZE,MOUNTPOINT,LABEL
