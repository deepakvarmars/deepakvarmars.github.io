matplotlib

By default, MacOS doesn't show the plots using plt.show().
Include following lines at the start:
include matplotlib matplotlib.use('TKAgg')

Warning for scipy

Using numpy.linalg.inv maynot provide the correct result using
python3 in Anaconda3 package manager. For eg, try
A = numpy.linalg.inv(numpy.matrix('1,2,3;4,5,6;7,8,9'))
The above is applicable to python version 3.6.3 in MacOS
