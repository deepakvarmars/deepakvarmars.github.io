Terminal Commands

sed

Stream editor
Used to search and replace particular string or to print all lines containing a string (similar to grep).
Examples:
To print lines containing string: sed -n '/u, v/p' log.txt > error.txt
To search and replace string : sed 's/oldstring/newstring/' log.txt > error.txt
To delete lines containing string: sed '/u, v/d' log.txt > newlog.txt

awk

Used to analyze data from data files organised in rows and columns
Examples:
To print specific columns (no. 3,4,5) : awk '{print $3,$4,$5}' log.txt > error.txt

grep

Used to get string in a particular command
Examples:
To get name of files in a folder and its subfolders containing specific string : grep -ilR "string" .
